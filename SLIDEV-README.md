# Presentación Slidev - Refactorización Spring PetClinic

## Requisitos

- Node.js 18+
- npm

## Ejecución rápida

Desde la raíz del proyecto:

```bash
npm init slidev@latest
```

Cuando termine el asistente:

1. Mantén los archivos generados de configuración
2. Reemplaza el contenido del `slides.md` generado con el `slides.md` de este repositorio
3. Ejecuta:

```bash
npx @slidev/cli slides.md --port 3030
```

## Exportar a PDF

```bash
npx @slidev/cli export slides.md
```

## Archivo principal

- `slides.md`: presentación completa (objetivos, casos de uso, proceso de refactorización, patrones y resultados)
